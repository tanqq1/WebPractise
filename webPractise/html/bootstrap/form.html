如何使用bootstrap创建表单

向父 <form> 元素添加 role="form"。 把标签和控件放在一个带有 class .form-group 的 <div> 中。 这是获取最佳间距所必需的. 向所有的文本元素 <input>、<textarea> 和 <select> 添加 class ="form-control" 。
内联表单 <form class="form-inline">
隐藏标签名<lable class="sr-only">
水平表单 <form class="form-horizontal">
lable添加class<lable class="control-lable">
表单控件：checkbox、input、textarea、radio、select
提供表单功能： 表单控件大小、表单控件状态（focus、success...）、静态控件（不可更改）、表单帮助文本
